# บริบทการทำงานปัจจุบัน (Active Context)

หน้านี้ระบุถึงจุดมุ่งหมายในปัจจุบัน สิ่งที่กำลังทำอยู่ และสถานะล่าสุดของโปรเจคในขณะนี้

---

## 1. สถานะปัจจุบันของโปรเจค (Current Status)
- **ระบบเสร็จสมบูรณ์พร้อมใช้**: ปัจจุบันหน้าจอแดชบอร์ดแสดงผลสรุปและข้อมูลรายรับ-รายจ่าย ปันผล รวมถึงภาษีได้ครบถ้วน
- **ระบบแก้ไขเสร็จสิ้นล่าสุด**: 
  - เพิ่มไอคอนลิงก์ภายนอก (`<i className="fa-solid fa-arrow-up-right-from-square">`) ถัดจากชื่อหุ้นในการ์ดสินทรัพย์ (StockCard), หน้าต่างป๊อปอัพอัปเดตข้อมูล (UpdateModal) และในป้าย Shariah (Musaffa)
  - ปรับปรุงปุ่ม "ปิดตา" และปุ่ม "รีเฟรชใหม่" ใน Header ของหน้าจอหลักให้มีขนาดเล็กลง กระชับ และพรีเมียมยิ่งขึ้น โดยลดขนาดฟอนต์ไอคอนใน JSX จาก `18px` เหลือ `14px` และปรับระดับ Padding ใน CSS จาก `0.625rem` เหลือ `0.45rem` พร้อมปรับระดับโค้งมน (`border-radius`) ให้เหลือ `0.5rem` เพื่อให้องค์ประกอบทุกส่วนสมดุลกันอย่างสวยงาม
  - ปรับแต่งตรรกะการเซนเซอร์ตัวเลขการเงินของการปิดตาข้อมูลในฟังก์ชัน `maskFormattedMoney` โดยปรับตัวหารความยาวตัวอักษรสำหรับสัญลักษณ์ดาวเด่น (`*`) จากเดิมหารด้วย `3` เปลี่ยนเป็นหารด้วย **`2.5`** เพื่อให้มีการปิดบังจำนวนหลักตัวเลขเพิ่มขึ้นและมีความปลอดภัยด้านข้อมูลส่วนบุคคลมากขึ้นเมื่อใช้งานโหมดปิดตา
- **การทดสอบและรันโลคอล**: เซิร์ฟเวอร์สำหรับพัฒนาถูกรันค้างไว้ด้วยคำสั่ง `npm run dev` บนเครื่องของผู้ใช้เพื่อดูผลลัพธ์แบบเรียลไทม์ และระบบมีความเสถียรดี

## 2. สิ่งที่กำลังโฟกัสในปัจจุบัน (Active Focus)
1. **การจัดทำระบบบันทึกความจำของนักพัฒนา (Developer Memory Bank)**:
   - จัดตั้งโครงสร้างโฟลเดอร์ `memory-bank` เพื่อให้นักพัฒนาร่วม (AI หรือคน) เข้ามาอ่านทำความเข้าใจสถาปัตยกรรมและฟีเจอร์เด่นของโปรเจคได้ในทันทีโดยไม่ต้องงมค้นหาโค้ดใหม่
   - จัดทำแผนภาพและเอกสารรายละเอียดของฟังก์ชันการแปลงอัตราแลกเปลี่ยน การคำนวณเป้าหมาย และระบบเครื่องคิดเลข Popover
2. **การรักษาคุณภาพโค้ดดั้งเดิม**:
   - ป้องกันและระวังไม่ให้การอัปเดตไฟล์สไตล์ CSS และตัวแปรฟังก์ชันใน `App.jsx` ส่งผลกระทบกับความลื่นไหลในการดึงข้อมูลจาก Google Apps Script

## 3. แผนการหรือโอกาสในการพัฒนาต่อยอด (Future Improvements)
แม้ว่าเว็บแอพจะมีประสิทธิภาพสูงในปัจจุบัน แต่มีบางจุดที่สามารถปรับแต่งเพื่อยกระดับระบบให้ดียิ่งขึ้นได้ในอนาคต:

### A. การปรับโครงสร้างโค้ด (Refactoring Component Separation)
*   **สถานะปัจจุบัน**: โค้ดทั้งหมด (1590+ บรรทัด) รวมอยู่ในไฟล์เดียวคือ `src/App.jsx`
*   **คำแนะนำ**: ในอนาคตสามารถแยกฟังก์ชัน/คอมโพเนนต์ย่อยออกเป็นไฟล์อิสระภายใต้โฟลเดอร์ `/src/components/` เช่น:
    *   `Calculator.jsx` (ระบบเครื่องคิดเลขจำลอง)
    *   `UpdateModal.jsx` (ป๊อปอัพฟอร์มแก้ไขข้อมูล)
    *   `StockCard.jsx` (การ์ดแสดงข้อมูลหุ้นรายตัว)
    *   `SummaryCard.jsx` (การ์ดสรุปยอดแดชบอร์ด)

### B. การแสดงผลกราฟสัดส่วนการลงทุน (Visual Charts)
*   เพิ่มคอมโพเนนต์การแสดงกราฟรูปวงกลม (Donut/Pie Chart) หรือกราฟแท่งเปรียบเทียบสัดส่วนของหุ้นแต่ละตัวในพอร์ตผ่านไลบรารีเบาๆ เช่น Recharts เพื่อช่วยให้การดูภาพรวมสัดส่วนการลงทุน (Asset Allocation) ทำได้รวดเร็วและสวยงามยิ่งขึ้น

### C. การเก็บข้อมูล Cache และ Offline Mode
*   เพิ่มระบบ Cache ข้อมูลลงใน `localStorage` ชั่วคราวหลังโหลดข้อมูลสำเร็จ เพื่อให้ผู้ใช้สามารถเปิดเว็บขึ้นมาดูข้อมูลล่าสุดได้ทันทีโดยไม่ต้องรอ API โหลดใหม่ทุกครั้งเมื่อเปิดเว็บ (ลดเวลาโหลดครั้งแรกและใช้ทำงานออฟไลน์ได้ชั่วขณะ)
